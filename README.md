# News Section Coding Test

コーディングテスト課題として、News（お知らせ）セクションをHTML / CSSで実装しました。

## Overview
- HTML5 / CSS3 で実装
- レスポンシブ対応
- BEMを意識したクラス設計
- セマンティックなマークアップ
- モバイル時はタイトルが2行を超える場合に省略表示
